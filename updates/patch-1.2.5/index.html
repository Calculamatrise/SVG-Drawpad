SVG Drawpad Patch Notes 1.2.5

Few changes, but hours behind the scenes.


New Features

Export/Import

You can now export/import your SVG Canvas via saving it to your computer!


Constant Colour Randomizer

A new feature in the advanced tools which automatically generates a new colour after a line is drawn.


Zoom Factor

Ctrl + (+/-) or scroll to zoom in/out.


New Tools

Curve Tool

Press down to lock your first anchor point, release to lock your second anchor point, and finally, decide the curving factor via moving your cursor and clicking on the canvas.


Dynamic Circle Tool

Uses various smaller lines to draw a circle from anchor point A to anchor point B.


New overlay style.

Improved the UI. That's about all I can remember. Thanks.
